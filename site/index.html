What Apache ActiveMQ Does


Apache ActiveMQ is an open source project that provides a multi-protocol asynchronous messaging system.

It can be run as standalone or embeddable, it has very high performance, ability to be clustered.

Apache ActiveMQ Artemis is fully JMS 2.0 spec compliant
Apache ActiveMQ Classic is JMS 1.1 compliant only.

MQTT
MQTT is a lightweight protocol, designed for constrained devices, low-bandwidth, high-latency or unreliable networks.
